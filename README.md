# totoro
totoro是一个前端工程化解决方案 名字源于宫崎骏动画片龙猫

# 依赖的模块
* uglify-js
* [stylus](https://github.com/LearnBoost/stylus "stylus")

## 主要功能
* 压缩js文件
* 发布
* js预编译
* 一键server
* 创建离线配置文件
* 基于stylus的css预编译
